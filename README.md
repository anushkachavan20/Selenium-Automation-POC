# Selenium Automation Framework (TestNG + Allure)

## Tech Stack
- Java
- Selenium 4
- TestNG
- Maven
- Allure Reports

## Features
- Page Object Model (POM)
- ThreadLocal WebDriver (Parallel Execution)
- Retry Analyzer for flaky tests
- Configurable browser & URL
- Screenshot capture on failure
- Allure reporting with attachments

## Run Tests
```bash
mvn clean test
